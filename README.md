## Introduction

Slang Graphics Library (sgl) is a modern, cross-platform graphics library written in C++ and Python.

## Installation

sgl is available as pre-compiled wheels via PyPI. Installing sgl is as simple as running

```bash
pip install nv-sgl
```

### Requirements

- `python >= 3.9`
